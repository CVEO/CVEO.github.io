# 视觉设计系统

## 配色
- 学术风格蓝白主色调
- Tailwind 定义品牌色 `brand`：
  - 500: #3b82f6
  - 600: #2563eb
  - 700: #1d4ed8

## 排版
- 使用 `@tailwindcss/typography` 渲染 Markdown 内容
- 基本文字颜色：`text-slate-800`，标题：`text-slate-900`

## 响应式
- 栅格采用 `grid` 与 `flex` 组合
- 断点：`sm`、`md`、`lg`、`xl`

## 交互
- 悬停态：`hover:shadow`、`hover:bg-brand-50`、`hover:text-brand-700`
- 平滑滚动：全局 `scroll-smooth`
- 过渡：`transition`、`duration-200`

## 照片墙
- 比例统一 3:4
- 点击放大使用模态层或 `dialog`
- 统一光照与浅色背景

## 组件
- 导航：固定顶部，半透明背景
- 卡片：圆角、边框、阴影
- 徽章：分区标注使用浅底深字

## 无障碍
- 图片均提供 `alt`
- 对比度满足可读性要求
