## 排版系统

### 字体
- **中文字体**: 系统默认字体栈
- **英文字体**: Inter, system-ui, sans-serif
- **代码字体**: Monaco, Consolas, monospace

### 文字颜色
- **正文**: `text-slate-800` (#1e293b)
- **标题**: `text-slate-900` (#0f172a)
- **次要文字**: `text-slate-600` (#475569)
- **链接**: `text-brand-600` (#2563eb)

### 字号与行高
- **小号**: `text-sm` (0.875rem) - 辅助文字
- **基础**: `text-base` (1rem) - 正文内容
- **大号**: `text-lg` (1.125rem) - 次要标题
- **标题**: `text-xl` (1.25rem) - 页面标题
- **大标题**: `text-2xl` (1.5rem) - 主要标题

## 响应式设计

### 断点系统
- `sm`: 640px - 移动设备
- `md`: 768px - 平板设备
- `lg`: 1024px - 桌面设备
- `xl`: 1280px - 大屏幕桌面
- `2xl`: 1536px - 超大屏幕

### 布局策略
- **栅格系统**: 使用 `grid` 与 `flex` 组合布局
- **容器宽度**: `max-w-7xl` (80rem) 最大宽度
- **间距系统**: 使用 Tailwind 标准间距比例

## 交互设计

### 悬停效果
- **卡片悬停**: `hover:shadow-lg hover:-translate-y-1 transition-all duration-200`
- **链接悬停**: `hover:text-brand-700 hover:underline`
- **按钮悬停**: `hover:bg-brand-600 active:bg-brand-700`

### 过渡动画
- **基础过渡**: `transition-all duration-200 ease-in-out`
- **平滑滚动**: 全局启用 `scroll-smooth`
- **页面切换**: 使用 Astro 内置的页面过渡

## 组件设计规范

### 导航栏 (NavBar)
- **位置**: 固定顶部
- **背景**: 半透明白色背景 `bg-white/95 backdrop-blur-sm`
- **高度**: `h-16` (4rem)
- **响应式**: 移动端汉堡菜单，桌面端水平导航

### 卡片组件
- **基础样式**: `rounded-lg border border-slate-200 bg-white shadow-sm`
- **悬停效果**: `hover:shadow-lg hover:-translate-y-1`
- **内边距**: `p-6` (1.5rem)
- **标题样式**: `text-xl font-semibold text-slate-900`

### 等高卡片布局
- **Footer卡片**: 4个等高卡片布局
  ```html
  <div class="flex flex-col items-center h-full">
    <div class="h-20 flex-shrink-0">...</div>
    <div class="flex-1 min-h-[2.5rem] flex items-center justify-center">
      <h3 class="line-clamp-2 leading-tight">...</h3>
    </div>
  </div>
  ```
- **合作单位卡片**: 等高网格布局
  ```html
  <div class="grid grid-cols-2 sm:grid-cols-3 md:grid-cols-4 lg:grid-cols-5 xl:grid-cols-6 gap-3">
    <div class="h-full">等高卡片</div>
  </div>
  ```
- **科研合作卡片**: 2列网格，含4:3图片
  ```html
  <div class="grid grid-cols-1 md:grid-cols-2 gap-4">
    <div class="aspect-[4/3] overflow-hidden rounded-lg">4:3图片</div>
  </div>
  ```

### 表格组件
- **表头**: `bg-slate-50 text-slate-700 font-semibold`
- **单元格**: `border-t border-slate-200`
- **悬停行**: `hover:bg-slate-50`
- **斑马纹**: 可选 `even:bg-slate-50`

### 徽章标签
- **分区标注**: `bg-blue-100 text-blue-800 text-xs font-medium px-2.5 py-0.5 rounded`
- **状态标签**:
  - 在研: `bg-emerald-100 text-emerald-800`
  - 已结题: `bg-slate-100 text-slate-800`
  - 已授权: `bg-blue-100 text-blue-800`

## 图片规范

### 成员照片
- **尺寸**: 统一 600×800 像素 (3:4 比例)
- **背景**: 浅色背景，统一光照
- **格式**: JPG 格式，优化压缩
- **命名**: `姓名-600x800.jpg`

### 内容图片
- **自适应**: 使用 `AdaptiveImage` 组件
- **懒加载**: 默认启用懒加载
- **替代文本**: 必须提供有意义的 `alt` 属性

## 无障碍设计

### 可访问性要求
- **图片描述**: 所有图片必须提供 `alt` 文本
- **颜色对比**: 确保文字与背景对比度 ≥ 4.5:1
- **键盘导航**: 支持完整的键盘导航
- **焦点指示**: 清晰的焦点状态指示器
- **语义化**: 使用正确的 HTML 语义标签

### ARIA 属性
- 为交互组件添加适当的 ARIA 属性
- 确保屏幕阅读器能够正确识别组件状态
- 提供必要的 ARIA 标签和描述

## 设计原则

### 一致性
- 保持组件样式和行为的一致性
- 使用统一的间距、颜色和字体系统
- 遵循 Tailwind CSS 设计规范

### 可用性
- 确保界面直观易用
- 提供清晰的视觉层次
- 优化移动端触摸体验

### 性能
- 优化图片加载性能
- 减少不必要的动画
- 确保快速的首屏加载时间
